O conceito de acessibilidade no senso comum, parece evidenciar os aspectos referentes ao uso dos espaços físicos. Entretanto, numa acepção mais ampla, a acessibilidade é condição de possibilidade para a transposição dos entraves que representam as barreiras para a efetiva participação de pessoas nos vários âmbitos da vida social. - Portal da UFC

A estruturação de uma página web precisa ser limpa, clara e intuitiva para que haja maior facilidade no acesso de todas as pessoas. Uma boa distribuição de conteúdo facilita o entendimento de usuários que precisam de leitores de tela, por exemplo.

A manutenção deve ser constante. O website deve estar sempre se atualizando e corrigindo erros que possam vir a surgir. É importante que disponibilize um canal de contato, para que usuários possam reportar erros.

Referências

  • Todo conteúdo não-textual deve ter uma alternativa em texto.

  • Imagens não decorativas devem conter um texto descritivo.

  • Aúdios devem ter transcrição.

  • Vídeo devem ter legenda ou intérprete de libras.

  • Textos devem ser alinhados a esquerda.

  • Deve ser espeitado a hierarquia de tag de cabeçalho.

  • Formulários devem ter rótulos para o identificar.

  • Links devem ser o próprio conteúdo.

  • Vídeos não devem ter autoplay.

  • Deve sempre haver uma opção de pausa.
